<h2 id="projeto">PROJETO</h2>

Site pessoal de links desenvolvido para centralizar meus principais canais, projetos e redes sociais em uma interface visual simples e animada.

Este foi meu segundo projeto autoral — desenvolvido para praticar HTML e CSS na construção de uma interface real, com foco em animações, microinterações e experiência visual.

🌐 [Acesse o site](https://lucaslinkverse.vercel.app/)

<h2 id="funcionalidades">FUNCIONALIDADES</h2>

- **Links centralizados** — acesso rápido a projetos, redes sociais e currículo
- **Animações interativas** — efeitos visuais em CSS nos elementos da interface
- **Background animado** — gradiente em movimento contínuo e infinito

<h2 id="tecnologias">TECNOLOGIAS</h2>

| Tecnologia | Uso |
|---|---|
| HTML5 | Estrutura semântica |
| CSS3 | Estilos, animações e layout |
| Git/GitHub | Versionamento de código |
| Vercel | Deploy do projeto |
| Canva | Prototipação e planejamento visual |
